#ChuckNorris

Цвет Чака Норриса (#chucknorris) — прикол в HTML-вёрстке и веб-дизайне, заключающийся в том, что при подстановке имени известного американского актёра Чака Норриса в качестве значения цветового кода в атрибутах тегов HTML, связанных с цветом, на веб-странице отображается тёмно-красный цвет.

Происхождение

Данный эффект был замечен пользователями в 2011 году на сайте Stack Overflow, посвящённом вопросам и ответам по программированию, где он приводился в качестве шуточного примера для иллюстрации особенностей интерпретации случайных наборов символов в качестве цветовых кодов в языке гипертекстовой разметки HTML. Результат подстановки произвольных букв и цифр в виде значений цветовых атрибутов приводил к необъяснимым эффектам и неожиданным цветам, что вызвало вопросы у многих пользователей. Код #chucknorris окрашивал элементы веб-страниц в тёмно-красный цвет, напоминающий цвет крови, что обыгрывало популярный мем про непобедимого героя боевиков Чака Норриса, убивающего всех врагов одним ударом.

Особенности определения цвета в HTML

Суть эффекта объясняется особенностью обработки ошибок при определении цветового кода в HTML, существующей ещё со времён браузера Netscape. При указании в качестве кода цвета произвольного набора символов происходит следующая последовательность преобразований: все буквы и символы, которые не входят в набор знаков шестнадцатиричной системы (т. е. цифр от 0 до 9 и букв A, B, C, D, E, F), заменяются нулями. Затем полученный набор символов дополняется нулями до количества знаков, делимого на три без остатка. Далее он разделяется на три части, в каждой из которых оставляются только первые два символа, а остальные отбрасываются.

В случае с именем Чака Норриса происходит следующее:

chucknorris → c00c0000000

c00c0000000 → c00c 0000 0000

c00c 0000 0000 → c0 00 00

В результате получается hex-код #C00000 или RGB (192, 0, 0), которому соответствует кроваво-красный цвет.

Добавление #chucknorris в качестве цветового кода работает только в атрибутах тегов HTML, связанных с цветом: в качестве значения для атрибута color тега font, атрибута bgcolor, задающего цвет фона элементов веб-страницы. В CSS подобный эффект уже не работает.
